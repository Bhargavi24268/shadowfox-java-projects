##  Technologies Used

* Java
* Object-Oriented Programming (OOP)

##  Project Files

### 1️ BankSystem.java

A console-based banking system that allows users to perform basic banking operations such as:

* Create account
* Deposit money
* Withdraw money
* Check account balance

Concepts used:

* Classes & Objects
* Methods
* Conditional Statements
* User Input Handling


### 2️ Contact_Managementsystem.java

A contact management application to store and manage contact details.

Features:

* Add new contacts
* View contact list
* Update contact information
* Delete contacts

Concepts used:

* Array/List handling
* OOP concepts
* Menu-driven program structure


### 3️ EnhancedCalculator.java

An advanced calculator application that performs multiple mathematical operations.

Features:

* Addition

### 4️ InventorySystem.java

A simple inventory management system for tracking items and stock.

* Classes and Objects

##  How to Run the Projects

1. Install Java (JDK 8 or above)
2. Compile the Java file:

   bash
   javac FileName.java
   
3. Run the program:

   bash
   java FileName

Example:

bash
javac BankSystem.java
java BankSystem
